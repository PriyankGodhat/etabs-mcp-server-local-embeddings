#### Parameters

Item

    Type: [SystemInt32](https://docs.microsoft.com/dotnet/api/system.int32)  
This is an integer between 1 and 9, inclusive, indicating the preference item
considered.

  1. Number of interaction curves
  2. Number of interaction points
  3. Consider minimum eccentricity
  4. Design for B/C Capacity Ratio?
  5. Gamma steel
  6. Gamma concrete
  7. Pattern live load factor
  8. Utilization factor limit
  9. Multi-response case design

Value

    Type: [SystemDouble](https://docs.microsoft.com/dotnet/api/system.double)  
The value of the considered preference item.

  1. Number of interaction curves 

Value >= 4 and divisable by 4

  2. Number of interaction points 

Value >= 5 and odd

  3. Consider minimum eccentricity 

0 = No

Any other value = Yes

  4. Design for B/C Capacity Ratio? 

1 = No

2 = Yes

  5. Gamma steel 

Value > 0

  6. Gamma concrete 

Value > 0

  7. Pattern live load factor 

Value >= 0

  8. Utilization factor limit 

Value > 0

  9. Multi-response case design 
     * 1 = Envelopes
     * 2 = Step-by step
     * 3 = Last step
     * 4 = Envelopes - All
     * 5 = Step-by step - All

#### Return Value

Type: [Int32](https://docs.microsoft.com/dotnet/api/system.int32)  
Returns zero if the item is successfully set; otherwise it returns a nonzero
value.
